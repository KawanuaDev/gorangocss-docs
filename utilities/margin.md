# Margin

{% hint style="info" %}
Since `1.0.0`
{% endhint %}

## Class

```css
.margin--{ auto | t | r | b | l | a }--{number}
```

```css
.m{ t | r | b | l | a }--{number}
```

## Code Sample

#### Margin top = 4 unit

```markup
<p class="mt--4 bg--gainsboro rounded--6 pa--1">
    Paragraph section with <kbd>.mt--4</kbd> or <kbd>.margin--t--4</kbd> class.
</p>
```
