# Float

{% hint style="info" %}
Since `1.7.0`
{% endhint %}

### Class

```css
.float--{left | right | none}
```

### Code Sample

#### .float--left

```markup
<p class="float--left">Float left!</p>
```

#### .float--right

```markup
<p class="float--right">Float right!</p>
```

#### .float--none

```markup
<p class="float--none">No float properties.</p>
```
